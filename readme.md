NekoCart
========

### Overview

Gameboy Flash Cartridge using CPLD as MBC.

### Project status

Working in progress.

### Hardware

Xilinx XC2C32A CPLD + 8MiB NOR (MX29LV640) + 128KiB SRAM (IS62LV1024)
