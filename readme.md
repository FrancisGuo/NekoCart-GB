NekoCart
========

### Overview

A Gameboy Flash Cartridge powered by STM32 instead of CPLD/FPGA.

### Project status

Suspended.

### Hardware

 * Main Processor: STM32F407VGT6
 * RAM: 192KB Internal + IS62LV1024LL SRAM (or FRAM alternative)
 * ROM: 1MB Internal + S29AL032D NOR
 * Expansion: TF card slot
