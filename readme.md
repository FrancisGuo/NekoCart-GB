NekoCart
========

# Overview

Open-source Gameboy Flash Cartridge using an Xilinx CPLD as MBC.

Visit ncgb.zephray.me for more information.

# Hardware

* Xilinx XC2C32A CPLD
* 8MiB NOR (MX29LV640)
* 128KiB SRAM (IS62LV1024)
